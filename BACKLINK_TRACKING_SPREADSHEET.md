## How to Use This Spreadsheet

### Setup Instructions:

1. **Copy to Google Sheets or Excel**
   - Create new spreadsheet
   - Create 9 separate sheets (tabs) for each section above
   - Copy tables into each sheet
   - Format with filters and conditional formatting

2. **Customize Formulas** (Google Sheets examples):
   - In Sheet 6 (Metrics), add formula for DA change: `=B2-B1`
   - Create dropdown menus for "Status" columns: Data > Data Validation
   - Add conditional formatting for priority levels (A+ = green, B = yellow, C = red)

3. **Weekly Routine:**
   - Monday: Review Sheet 1 (Opportunities), select 5-10 to pursue this week
   - Daily: Update Sheet 2 (Outreach), Sheet 5 (HARO)
   - Friday: Complete Sheet 8 (Weekly Log)
   - End of Month: Update Sheet 6 (Metrics)

4. **Monthly Review:**
   - Analyze Sheet 6 for trends
   - Adjust priorities in Sheet 1 based on results
   - Plan next month's focus areas
   - Update budget in Sheet 9 if needed

### Color Coding Suggestions:

- **Green:** Completed/Acquired
- **Yellow:** In Progress
- **Red:** Rejected/Not Viable
- **Blue:** High Priority
- **Gray:** Low Priority/Paused

### Automation Ideas:

- Use Zapier to auto-log Google Alerts to Sheet 2
- Connect Ahrefs API to auto-update Sheet 6 metrics
- Set up Google Sheets notifications for weekly reminders
- Create dashboard view with charts from Sheet 6
